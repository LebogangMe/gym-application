# 🏋️ Gym Management Website

## 📌 Description
This project is a modern and responsive Gym Website developed to provide users with an easy way to explore fitness services, view membership plans, and stay informed about health and wellness.

The goal of this project is to deliver a clean, user-friendly interface that enhances user experience and promotes fitness engagement.

## 🚀 Features
- 🧾 View gym membership plans and pricing
- 🏋️ Explore workout and fitness information
- 📱 Fully responsive design (mobile, tablet, desktop)
- 🎯 Clean and intuitive user interface
- 📩 Contact or enquiry section (if you added one)

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript
- React

- 🚀 Live Demo: https://lebogangme.github.io/gym-application/
